Das Werk



Mood-Tracker :

Beim Öffnen der App wird dem Anwender zunächst der Mood-Tracker angezeigt. (Voraussetzung ist, das der Nutzer bereits ein Konto erstellt hat.) Es werden dem User 5 Herzen angezeigt.
Die Herzen stehen für:
❤️‍🔥 = sehr gut,
❤️ = gut,
🧡 = okay,
❤️‍🩹 = nicht so gut,
💔 = schlecht.
Nachdem der Anwender seine Stimmung ausgewählt hat, wird im passend zu seiner Stimmung, ein Spruch angezeigt. Diesen Spruch kann man über ein Like speichern.


Tagebuch:

Es gibt die Kategorie „Tagebuch“. Hier gibt es vier Tagebücher. Sobald ein Tagebuch ausgewählt wurde und auf das Datum geklickt wurde, wird man zu der Kalenderansicht geleitet. Hier kann man aufgrund kleiner farbigen Punkte sehen, an welchen Tagen, für welches Tagebuch etwas eingetragen wurden. Denn jeder kleine Punkt hat die entsprechende Farbe des zugehörigen Tagebuchs. Auch kann man die vergangenen Tagebucheinträge lesen. Für ein neuen Eintrag wählt man den aktuellen Tag aus geht dann auf den Button „zurück“. Nun wird der Anwender gefragt, ob er eine Vorlage benutzen möchte oder lieber ein leeres Blatt haben möchte.


Aufgaben und Challenges :

Unter Kategorie „Aufgaben und Challenges“ können verschiedene Aufgaben und Challenges bearbeitet werden, die das Selbstbewusstsein steigern sollen. Es gibt für beide Reiter 3 Levels; Beginner, Fortgeschritten und Profi. Für jedes Level gibt es eine Anzeige, die dem Anwender anzeigt, wie viel Prozent von dem entsprechenden Level schon absolviert wurde. Sobald 100% erreicht sind, steigt der Nutzer ein Level auf.
